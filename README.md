# Channel Playlists Calculator

A script to analyze YouTube channel playlists and generate detailed statistics.

## Features

- Extracts all playlists from a YouTube channel.
- Calculates total videos, average video length, and total duration for each playlist.
- Saves the analysis to a file (`playlists_info.txt`).

## How to Use

1. Update the `channel_url` variable with the desired YouTube channel's URL.
2. Run the script: `python channel_playlists_calc.py`
3. View the results in the `playlists_info.txt` file.

## Requirements

- Python 3.7+
- `yt_dlp`

Install dependencies using:

```bash
pip install yt-dlp
```
